四、 js组成和命名规范

分享
  • 层级有关系
  • 命名规范
  • 关键字和保留字

  • 一、层级有关系：

    1.浏览器(window浏览器对象)--->文档(document文档对象)--->html--->head/body--->...

  • 二、js: 是javascript, 是一门轻量级的脚本编程语言：

    1.ECMAScript(4、5)：定义了js里面的命名规范、变量、数据类型、基本语法，操作语句等最核心的东西

    2.DOM:document object model 文档对象模型

  • 三、命名规范：

    1.js中严格区分大小写 test Test

    2.使用驼峰命名法

    	-2. 可以使用数字、字母、下划线、$(数字不能作为首位)
		var _get = "获取";

    2.关键字和保留字

    	-3. 不能使用关键字和保留字
		关键字：在js中有特殊意义的字
		保留字：未来可能成为关键字的
