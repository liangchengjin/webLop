  • 一、基本数据类型和引用数据类型的区别

    -1.腾讯面试题

    
var num1 = 12;
var num2 = num1;			// 把num1变量代表的值给了 num2 变量 
num2++;					// 相当于 num2 = num2+1 在自己原有值的基础上加1 也可能写成 num2+=1;
console.log(num2);


var obj1 = {name:"赵子龙"};
var obj2 = obj1
obj2.name = "常山赵子龙(赵云)";
console.log(obj1.name);

// 发现：基本数据类型没有跟着改变，引用数据类型跟着也改变了

  • 二、基本数据和引用数据类型的本质区别：

    一句话来说明：基本数据类型操作的是值，而引用数据类型操作的是对新空间的引用地址

    -1.基本数据类型是把值直接的给变量，接下来在操作的过程中，直接拿这个值操作的，可能两个变量存储一样的值，但是你的是你的，我的是我的，之间没有关系，基本一个改变，另一个没有任何的影响

    引用数据类型：

    
			
1).定义一个变量
2).开辟一个新的空间，然后把属性名和属性值保存在这个空间中，并且有一个空间地址
3).把空间的地址给了这个变量，变量并没有存储这个数值，而是存储的是对这个空间引用地址
4).接下来我们把这个地址，又告诉给了另外一个变量，另外一个变量存储的也是这个地址，此进两个变量操作的是同一个空间
5).其中一个改变了的内容，另外一个也跟着改变了

//赵子龙
//李连杰

// 基本数据类型：
	赵子龙开了一个淘宝店，卖的是吃的(湖南长沙臭豆腐)
	李连杰也开了一个淘宝店，卖的和赵子龙店里的东西一样

// 引用数据类型：
	赵子龙开了一个淘宝店，卖的是吃的(湖南长沙臭豆腐)
	赵子龙有一个店的账号地址
	李连杰拿到了赵子龙的账号址址

    -1.type 是用来检测数据类型的:typeof 要检查的值

    typeof null的结果是 "object"

    

			typeof 1			//	"number"
			typeof true			//	"boolean"
			typeof undefined		//	"undefined"
			typeof null			//	"object"
			typeof {}			//	"object"
			typeof []			//	"object"
			typeof /^$/			//	"object"

			typeof 的局限性：不能具体的检查object下细分的类型，检查这些返回的都是"object"
				typeof function(){}				//	"function"

				typeof function(){}				//	 "function"

		
				console.log(typeof("呈琛"));			// "string"
				var num2 = 12;
				console.log(typeof num2);			// number 把num2代表的值进行检测，检测是什么数据类型在

		  一秒中回答，超过1秒钟后，就不能成功面试
			console.log(typeof (typeof (typeof (typeof[]))));	// "string" 出现两个和两个以上的typeof的时候，最终结果都是"string"
			
				typeof []		===> "object"
				typeof "object"	===> "string"
				typeof "string" ===> "string"
