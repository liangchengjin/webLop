三、js中常用的四种输出方式

分享
  • js中的常用的几种输出方式
  • js 引入在html页面里有几种样式呢

  • 一、js中的常用的几种输出方式：

    四种

    	
    1.alert("内容");			说明：在浏览器弹出框显示我们的内容
    
	alert("内容");

	
    2.document.write("内容"); 说明： 在页面输出显示我们的内容
    	
	document.write("欢迎大家来到深圳");

    
    3.console.log("内容");	说明：最常用的一种方式，在控制台输出我们的内容 可以按 F12打开控制台 console的页卡中查看我们的输出的内容(谷歌浏览器)	
    
	console.log("我叫呈琛！我的QQ号是：553585163");

	
    4.innerHTML / innerText: 向指定元素中动态添加内容
