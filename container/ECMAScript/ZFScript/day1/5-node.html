五、 js中的变量和数据类型

分享
  • css 引入在html 页面中的几种方式
  • js 引入在html页面里有几种样式呢

  • 一、变量：可变的量：

    1.js中的变量是一个抽象的概念，变量是用来存储值和代表值的

  • 二、变js中定义一个变量非常的简单：

    1. var 变量名 = 变量值;

    2. = 是赋值操作， 左边的变量名 右边是存储的值，

    3. js中的变量是松散类型的：通过一个var 变量就可以存储的数据类型

    	
			var name = "呈琛"; // 定义一个变量 name，把字符串"呈琛"赋值给这个变量
			console.log(name); // 把 name 代表的值输出在控制台

  • 三、js中的数据类型分类：

    
var num = 12;
var str = "深圳";			// 字符串
var boo = true;				// true false	
var n = num;
var m = undefined;

var obj = {name:"chengcheng"};		// {} 对象数据类型
var ary = [1,2,3,4];			// [] 数组 也是对象数据类型
var reg = /^1\d{11}$/;			// 正则 也是对象数据类型	
var fn = function(){ };			// 函数数据类型

    1. 基本数据类型：由简单的结构的数字(number)、字符串(string)、布尔类型(boolean)、null、undefined

    2. 引用数据类型:结构相对复杂一些的,包含了：

    
对象数据类型(object)
函数数据类型(function)
