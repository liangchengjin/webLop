  • 一、1.js数据类型中的对象数据类型

    -0.js中对象数据类型的特点，是由多组 [属性名和属性值] 组成，多组键值组成，由多个key:valu

    
var obj = {name:"呈琛", age:6};				// name是属性名 "呈琛"是属性值
		console.log(obj.name);
		console.log(obj.age);

		var personInfo = {
			name:"赵子龙",
			age:28,
			height:"180cm",
			weight:"60kg"
		}
		console.log(personInfo.name);		// 赵子龙
		console.log(personInfo.age);		// 28
		console.log(personInfo.height);	// 180cm
		console.log(personInfo.weight);	// 60kg

		var obj = {name:"深圳"};		// 这种方式叫做 字面量创建方式，是可以看到的
		var obj = new Object();		// 这种方式叫做 实例创建的方式，这里面是没有放任何的东西的。

    -1.给一个实例的对象 增加一组属性名和属性值

    
obj.name = "cc";		// 第一种增加一个属性名和属性值
obj["name"] = "cc";		// 第二种增加一个属性名和属性值
obj["age"] = 6;

    -2.修改原有属性名的属性值，规定一个对象中的属性名不能重复，如果之前有就是修改，如果没有就是增加

    
obj["name"] = "呈琛";		// 第一种修改属性名的属性值
obj.name = "呈琛";		// 第二种修改属性名的属性值

    -3.获取属性名和属性值,如果属性名不存在，默认返回的结果是undefined

    
console.log(obj["name"]);	// 呈琛
console.log(obj.name);		// 呈琛
console.log(obj.zz);		// undefined

    -4.删除属性名和属性值

    
		
obj.age = null;		// 假删除
console.log(obj);	// Object {name: "呈琛", age: null}

delete obj.age;		// 真删除
console.log(obj);	// Object {name: "呈琛"}

  • 三、js中对象、类、实例的区别：对象是泛指，js中万物皆对象，类是对对象的具体的细分，实例是类中的一个具体的事物

    -1.举例：

    自然界中万物皆对象，所有的东西可以分为：人类、植物类、动物类、物体类等......，每一个人都是类中的一个具休中的实例
