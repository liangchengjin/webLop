    0.学习方法

    1. 第一组：关于数组的增加、删除、修改

    1). push 向数组的末尾添加新的内容, 返回添加后新数组最终的长度改变了

    
var ary = [10,12,13,11,19,17];
var res = ary.push(18,20);  
console.log(res);	// 7

    2). unsift 向数组的开头增加新的内容，返回的也是增加后的数组的长度，原来的数组也改变了

    
var ary = [10,12,13,11,19,17];
var res = ary.unsift(16.22);	// 7
console.log(ary);
console.log(res);

    3-0). splice(n, 0, x); 向数组的中间某个位置添加新的内容 从索引n开始，删除0个内容，把新增加的内容x放在索引n的前面，返回的是一个空数组，原有的数组改变

    
var ary = [10,12,13,11,19,17];
var res = ary.splice(2, 0, "呈琛");
console.log(ary);
console.log(res); //[]

    3-1). splice(n, m); 删除数组中指定的某些项，坐过引n开始(包含n), 删除m个元素, 把删除的内容当做一个新的数组返回，原有的数组改变

    
var ary = [10,12,13,11,19,17];
var res = ary.splice(2, 2);
console.log(ary);  // [10, 12, 19, 17]
console.log(res);  // [13, 11]

    3-2). splice(n, m, x); "替换" 把原有数组中的某些项进行替换(删除，然后用x替换)从索引 n开始删除m个元素，用x替换原来的，把删除的内容当做一个新的数组返回，原有数据改变

    3-3).规律：

    
 规律：splice(0, 0, x) 		 相当 于我们的unshift 
	splice(ary.length, 0, x)				 相当于我们的push

    3-3). 应用splice

    
var ary = [10,12,13,11,19,17];
//var res = ary.splice(2,3,"中国深圳");
//console.log(ary);  // [10, 12, "中国深圳", 17]

var res = ary.splice(ary.length, 0,166);
console.log(res);
console.log(ary);  // [10, 12, 13, 11, 19, 166, 17]

    3-4).4). pop(); 删除数组最后一个，返回的是删除那一项，原有数组改变

    3-5).5) shift 删除数组的第一个，返回的是删除的那一项，原有数组改变

    2. 第二组：数组的查询和复制

    1). slice(n, m); 坐索引n开始(包含n),找到索引m处(不包含m),然后把找到的内容做为一个新的数组返回，原有的数组是不改变的

    
var ary = [10,12,13,11,19,17];
var res = ary.slice(2, 5);
console.log(ary);  // [10, 12, 13, 11, 19, 17]
console.log(res);  // [13, 11, 19]

    2). slice(n); 坐索引n开始(包含n)，查找到数组的末尾，然后把找到的内容作为一个新的数组返回，原有的数组是不改变的

    slice(0)或者slice() 将原有的数组原封不动的复制一份，属于数组的克隆 

    
var ary = [10,12,13,11,19,17];
var res = ary.slice(0);
console.log(ary); // [10, 12, 13, 11, 19, 17]
console.log(res); // [10, 12, 13, 11, 19, 17]

    3).concat(); 也可以实现数组的克隆,原来的数组也不改变(相当于slice(0))

    说明：其实它不是真的克隆，它只是数组拼接(连接)

    
var ary = [10,12,13,11,19,17];
var res = ary.concat();
console.log(ary);  // [10, 12, 13, 11, 19, 17]
console.log(res);  // [10, 12, 13, 11, 19, 17]

    4).concat 的本意是实现数组的拼接的， ary.concat(ary2) 将数组ary和ary2拼接成一个数组，原来 的数组也不改变，这就是concat

    
var ary = [10,12,13,11,19,17];
var ary2 = ["深圳","12k"];
var res = ary.concat(ary2);
console.log(ary);  // [10, 12, 13, 11, 19, 17]
console.log(res);  // [10, 12, 13, 11, 19, 17, "深圳", "12k"]

    2).join(分隔符) 把数组中的每一项拿出来用指定的分隔符隔开，原有数组不变

var res = ary.join("+");			
console.log(ary);   // [10, 12, 13, 11, 19, 17] 
console.log(res);   // 10+12+13+11+19+17

    4. 第四组：数组的排列和排序

var res = ary.reverse();			
console.log(ary);	// [17, 19, 11, 13, 12, 10]			
console.log(res);	// [17, 19, 11, 13, 12, 10]
